# Burrito King Restaurant Application

## IDE Used
IntelliJ IDEA 2024.1

## Java Version
OpenJDK 22

## JavaFX Version
JavaFX 22-ea+11

## Database Used
No database used; data is stored in `.dat` files.

## Installation and Running Instructions

1. **Clone the Repository:**
   ```
   git clone [YOUR_GITHUB_REPOSITORY_URL]
   ```

2. **Open in IntelliJ IDEA:**
   - Open IntelliJ IDEA.
   - Click on `File` -> `Open`.
   - Navigate to the cloned repository folder and select it.

3. **Setup JavaFX:**
   - Ensure JavaFX SDK is set up in your project.
   - Go to `File` -> `Project Structure` -> `Libraries` and add your JavaFX SDK path.

4. **Run the Application:**
   - Open `HelloApplication.java` in IntelliJ.
   - Click the green run button in the top-right corner.
